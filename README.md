AnimatedList Widget
A minimal Flutter app demonstrating AnimatedList which is a scrollable list widget that automatically animates items as they are inserted or removed.

Run:
flutter create animatedlist_widget
cp animated_list_demo.dart animatedlist_widget/lib/main.dart
cd animated_list_demo && flutter run
